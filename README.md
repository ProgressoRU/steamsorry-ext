# Steam: Ой, извините!

Расширение для Chrome и Firefox (Manifest v3), которое восстанавливает видимость недоступных в России Steam игр. Также может работать внутри браузера нативного клиента Steam через [Milleniumn](https://docs.steambrew.app/) и 
[Extendium](https://steambrew.app/plugin/788ed8554492). 

Расширение выполняет несколько функций:

1. **Перенаправление со страниц магазина.** Когда расширение обнаруживает, что пользователь попал на страницу с ошибкой  ("Ой, извините!"), оно инициирует перезагрузку страницу из под другого региона с помощью параметра ?cc. Т.к. Steam игноирует этот параметр для авторизованных пользователей, то расширение перед навигацией скрывает авторизационную куки. В обычных браузерах это достигается с помощью простого declarativeNetRequest правила. При использовании внутри клиента Steam используется более сложный (и подверженный ошибкам) способ с временной подменой и восстановлением куки.

2. **Подмена результатов поиска.** Расширение перехватывает запросы `SearchSuggestions` из магазина и переписывает параметр `context.country_code`. Таким образом достигается возвращение скрытых игр в быстрый поиск по магазину. Страница "продвинутого" поиска также автоматически подгружается с подменой куки (как описано в п.1), а AJAX запросы с нее перехватываются и подменяются на неавторизованные. 

3. **Обход проверки возраста.** Т.к. с проверкой возраста при манипуляциях с авторизационной куки возникали проблемы в расширение также встроен механизм обхода этой проверки. (При запуске добавляет куки, которые дают магазину понять, что проверка уже пройдена)

## Build

Требует [Bun](https://bun.sh).

```sh
bun install
bun run build
```
